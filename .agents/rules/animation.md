Animation Philosophy

1. Every animation should feel premium and intentional.

2. Use spring animations instead of linear transitions.

3. Avoid animations under 0.4 seconds.

4. Add stagger effects wherever multiple elements appear.

5. Add subtle floating motion to photos.

6. Use parallax on scroll sections.

7. Use smooth scrolling via Lenis.

8. Use GSAP ScrollTrigger for scroll storytelling.

9. Keep frame rate above 60fps.

10. Respect prefers-reduced-motion.
